Simple graphics library

The library contains wrappers around basic primitives from the SDL_gfx library. It is intended to provide basic graphics capabilities to the beginners learning C. It is expected, that after getting enough proficiency in C, the programmer will use SDL library directly. The library is contained in the file primlib.c. Every file using it should include primlib.h. The library can be downloaded here. It requires SDL 2.0 and SDL2_gfx, which can be installed in Ubuntu using the following command:

sudo apt install libsdl2-gfx-dev

The coordinate system

The coordinate system origin is located at upper-left corner of a graphics window. Width and height of the graphics window can be queried using the gfx_screenWidth() and gfx_screenHeight() functions, respectively. Before using any of the graphics functions gfx_init() must be called. After the drawing operations, to make the updates to the screen visible, call gfx_updateScreen(). Before calling gfx_updateScreen() once again, you must clear the screen and redraw the entire scene.

Compilation

The following command line can be used to compile programs using the library:

gcc -g -Wall -pedantic `sdl2-config --cflags` primlib.c progname.c -o $@ -lSDL2_gfx `sdl2-config --libs`

where progname.c is the name of the main program module.

Function reference

int gfx_init();

Must be called before any graphic operation. Returns 0 on success, non-zero on failure.

enum color {BLACK, RED, GREEN, BLUE, CYAN, MAGENTA, YELLOW, WHITE, COLOR_MAX};

Enumeration type, representing available colors.

void gfx_pixel(int x, int y, enum color c);

Put pixel with color c at coordinates (x,y).

void gfx_line(int x1, int y1, int x2, int y2, enum color c);

Draw line from (x1,y1) to (x2,y2), using color c.

void gfx_rect(int x1, int y1, int x2, int y2, enum color c);

Draw rectangle with upper left corner at (x1,y1) and lower right corner at (x2,y2), using color c.

void gfx_filledRect(int x1, int y1, int x2, int y2, enum color c);

Draw filled rectangle with upper left corner at (x1,y1) and lower right corner at (x2,y2), using color c.

void gfx_filledTriangle(int x1, int y1, int x2, int y2, int x3, int y3, enum color c);

Draw filled triangle with vertex coordinates (x1,y1), (x2,y2) and (x3,y3), using color c.

void gfx_circle(int x, int y, int r, enum color c);

Draw circle with a center at (x,y) and radius r, using color c.

void gfx_filledCircle(int x, int y, int r, enum color c);

Draw filled circle with a center at (x,y) and radius r, using color c.

void gfx_textout(int x, int y, char *s, enum color c);

Print text s starting at the coordinates (x,y), using color c.

int gfx_screenWidth();

Get the width of the screen.

int gfx_screenHeight();

Get the height of the screen.

void gfx_updateScreen();

Makes the result of drawing functions since the last update visible on the screen.

int gfx_pollkey();

Return the code of pressed key, do not wait for the keypress. Return-1 if no key has been pressed.

int gfx_getkey();

Wait for the keypress and return the key code.

int gfx_isKeyDown(int key);

Return 1, if the key key is pressed, 0 otherwise.
